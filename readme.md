A space-survival game where you control a spaceship and dodge incoming asteroids. The goal is simple: survive as long as possible and 
achieve the highest score. Best for fun and implimenting basic game development concepts.

# Features:
	1. Smooth spaceship movement 
	2. Randomly creating asteroids 
	3. Score increase over time.
	4. Game over screen
	5. Clean UI and simple control
	6. Lightweight and easy to run
	7. Spaceship can shoot lasers
	8. Large Asteroids splits into two smaller asteroids.
	9. Increasing more diffuilty when fragment increases.


# How to play:
	1. Press Enter to start the game
	2. Use arrow keys to control the spaceship
	3. Avoid asteriods
	4. Press Space to shoot laser
	5. Shooting big asteroid -> breaks into teo smaller piece
	6. Shooring small asteroid again -> Destroy that completely
	7. Avoid any crash
	8. Survive as long as possible to get higher score
